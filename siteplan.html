WDD 131 | Personal Web Developer Portfolio

Site Name

Site Name: thetobsdev

The name thetobsdev represents my personal brand as a web developer. It is short, modern, and professional, making it easy to remember and suitable for showcasing my skills, projects, and experience. The name clearly communicates that the website belongs to a developer.

Optional domain availability: thetobsdev.dev

Site Purpose

The purpose of this website is to serve as a professional portfolio that presents my web development skills, tools, and completed projects. The site will provide information about my technical abilities, display my work through a project gallery, and allow visitors to contact me for job opportunities or freelance work.

Scenarios

  • What web development projects has this developer worked on?
  • What programming languages and tools does this developer use?
  • How can I contact this developer for collaboration or employment?

Color Schema

The following color schema will be used throughout the website:

  • Dark Blue (#0A2540): Used for headers, navigation bar, and footer.
  • Light Gray (#F4F6F8): Used for page backgrounds and content areas.
  • Gold Accent (#F5A623): Used for buttons, highlights, and accents.

Typography

  • Montserrat: Used for headings and navigation elements.
  • Open Sans: Used for body text to improve readability.
